## Tentang Proyek

Aplikasi **Augmented Reality Bangunan Bersejarah Pulau Jawa** merupakan media edukasi interaktif yang memungkinkan pengguna **menjelajahi bangunan bersejarah dalam bentuk model 3D** hanya dengan melakukan **pemindaian marker atau QR Code** menggunakan smartphone Android.

Melalui teknologi AR, bangunan bersejarah tidak hanya ditampilkan sebagai gambar, tetapi hadir secara visual di dunia nyata, lengkap dengan **informasi sejarah singkat** yang mudah dipahami.

Sebagai pelengkap, **WebRMDB** berfungsi sebagai:
- **Landing page resmi aplikasi**
- **Media informasi alternatif** apabila perangkat pengguna tidak mendukung AR
- Sarana pembelajaran sejarah digital berbasis web

Dengan tampilan **klasik bernuansa heritage**, WebRMDB dirancang agar selaras dengan tema budaya dan sejarah yang diangkat.
