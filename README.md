# Bangunan Bersejarah Pulau Jawa – WebRMDB

**WebRMDB** adalah website edukasi yang memperkenalkan **bangunan bersejarah terkenal di Pulau Jawa** sebagai bagian dari proyek **Augmented Reality (AR) edukatif**. Proyek ini menggabungkan teknologi AR dan media web untuk menghadirkan pengalaman belajar sejarah yang **interaktif, modern, dan mudah diakses**.

---

## Tentang Proyek

Aplikasi ini merupakan **aplikasi Augmented Reality (AR)** yang dirancang untuk memperkenalkan bangunan bersejarah di Pulau Jawa sebagai **media pembelajaran interaktif**. Melalui teknologi AR, pengguna dapat menampilkan **model 3D bangunan bersejarah** beserta **informasi sejarah singkat** dengan cara melakukan **pemindaian marker atau QR Code**.

Website **WebRMDB** berfungsi sebagai **media pendukung dan alternatif akses informasi**, khususnya ketika perangkat pengguna **tidak mendukung fitur Augmented Reality**. Meskipun tanpa AR, pengguna tetap dapat mengakses informasi sejarah bangunan secara digital melalui **QR Code** yang tersedia pada kartu.

---

## Fitur Utama
- Informasi edukatif bangunan bersejarah di Pulau Jawa  
- Media pendukung aplikasi Augmented Reality  
- Akses informasi melalui **QR Code**  
- Tampilan **klasik bernuansa heritage**  
- Alternatif pembelajaran sejarah berbasis web

## Unduh Aplikasi

Aplikasi **Augmented Reality Bangunan Bersejarah Pulau Jawa** dapat diunduh melalui tautan yang tersedia pada website ini. Aplikasi dirancang khusus untuk **perangkat Android** dan memanfaatkan kamera serta sensor perangkat untuk menjalankan fitur AR secara optimal.

### Persyaratan Minimum
- Sistem Operasi: **Android 11** atau versi lebih baru  
- Kamera dan sensor perangkat mendukung AR  
- Ukuran file aplikasi: **± 202 MB**
